MagicCubiano
=============
Source code of app for playing music on Xiaomi Giiker Cube.

## Description

Using Qt 5.9.5. Testing on Adroid only.

## Knowing issues

Add midi support;
Add background mode;
Add "Turn on bluetooth!" message;
Reduce connection time with a large number of BLuetooth devices nearby;
